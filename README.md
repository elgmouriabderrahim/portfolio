# Abderrahim Elgmouri Portfolio

Premium Vue 3 portfolio for Abderrahim Elgmouri, built with Vite, Tailwind CSS, GSAP, Lenis, and Three.js.

## Features

- Immersive animated hero with lazy-loaded Three.js scene
- GSAP + ScrollTrigger reveal system
- Smooth scrolling with Lenis
- Dark futuristic visual system with glassmorphism and neon accents
- Featured projects backed by real CV data
- Live GitHub repository section via public API
- Timeline, skills grid, and contact form

## Development

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
